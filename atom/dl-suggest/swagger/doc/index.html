DL Suggest

Base URL: /v1, Version: 1.0.0

入力されたコードに対して次に予測されるコードの候補を返却。

daemonの起動

    $ bin/dlsuggestd -p [PORT_NUM] \
                     -m [MODEL_FILE] \
                     -w [WORD_TO_IDX_FILE] \
                     -t [TARGET_TO_IDX_FILE]

PORT_NUM: tcpのポート番号。デフォルト: 9999 MODEL_FILE: 学習済みのモデルファイル WORD_TO_IDX_FILE: word_to_idx.npy TARGET_TO_IDX_FILE: target_to_idx.npy

daemonの終了

    $ bin/dlsuggestd close

daemonのステータス

    $ bin/dlsuggestd status
